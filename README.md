# CineLog 🎬

Este es mi proyecto final del bootcamp BTech. Es una app para guardar peliculas que quiero ver y las que ya vi.

## Que hace la app?

- Buscas una pelicula y te trae los datos de la API de OMDB (titulo, poster, descripcion)
- La agregas a tu lista y queda guardada en Firebase
- Podes marcarla como "Ya la vi" o dejarla como "Por ver"
- Tambien podes editar los datos o eliminarla
- Las peliculas se guardan en Firestore asi que si recargás la pagina siguen ahi

## Tecnologias que use

- React (con Vite)
- Firebase / Firestore
- API de OMDB
- react-icons
- CSS puro

## Como correr el proyecto

1. Clonar el repositorio
2. Instalar las dependencias:

```
npm install
```

3. Correr el proyecto:

```
npm run dev
```

4. Abrir en el navegador la url que te da la terminal
